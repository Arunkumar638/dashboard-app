Keywords

In JavaScript, there are a number of reserved keywords that have special meanings and purposes in the language. These keywords cannot be used as variable names, function names, or identifiers. Here is a list of some of the most commonly used keywords in JavaScript:


  1. break: Used to exit a loop prematurely.
  2. case: Used in a switch statement to specify different cases.
  3. catch: Used to handle exceptions in a try...catch block.
  4. class: Used to define a class in JavaScript (ES6 feature).
  5. const: Used to declare a constant variable that cannot be reassigned.
  6. continue: Used to skip the rest of the loop and continue with the next iteration.
  7. debugger: Used to stop the execution of JavaScript code and launch the debugger.
  8. default: Used in a switch statement to specify the default case.
  9. delete: Used to delete properties from an object.
  10. do: Used to create a do...while loop.
  11. else: Used in an if...else statement.
  12. export: Used in module systems to export values from a module.
  13. extends: Used to extend a class in a class declaration.
  14. finally: Used in a try...catch...finally block to specify code that will be executed regardless of whether an exception is thrown or caught.
  15. for: Used to create a for loop.
  16. function: Used to define a function.
  17. if: Used in an if statement to specify code that will be executed if a condition is true.
  18. import: Used in module systems to import values from other modules.
  19. in: Used to check if a property exists in an object.
  20. instanceof: Used to check if an object is an instance of a particular class or constructor.
  21. let: Used to declare a block-scoped variable (introduced in ES6).
  22. new: Used to create an instance of a constructor function.
  23. return: Used to return a value from a function.
  24. super: Used to call functions on an object's parent.
  25. switch: Used to create a switch statement.
  26. this: Refers to the current object.
  27. throw: Used to throw an exception.
  28. try: Marks a block of statements to try, and specifies a response, should an exception be thrown.
  29. typeof: Used to determine the type of a value.
  30. var: Used to declare a variable (pre-ES6; replaced by let and const)
    31. .
  31. void: Specifies that a function does not return any value.
  32. while: Used to create a while loop.
  33. with: Introduces a block of code with a specified object as the scope.
  34. yield: Pauses and resumes a generator function.

Please note that this is not an exhaustive list, but it covers many of the fundamental keywords in JavaScript. Always be mindful of using these words for variable or function names, as doing so will result in a syntax error.
