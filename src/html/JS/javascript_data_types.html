DataTypes:

In JavaScript, there are several data types that can be categorized into two main categories: primitive and non-primitive (also known as reference) data types.


Primitive Data Types:

Number: Represents both integer and floating-point numbers.

let num = 42; // Integer

let floatNum = 3.14; // Floating-point number

String: Represents a sequence of characters (text).

let str = "Hello, World!";

Boolean: Represents a true or false value.

let isTrue = true;

let isFalse = false;

Undefined: Represents a variable that has been declared but hasn't been assigned a value yet.

let undefinedVar;

Null: Represents the absence of any value or object.

let nullValue = null;

Symbol (ES6): Represents a unique and immutable value, primarily used as object property keys.

const uniqueSymbol = Symbol('description');

BigInt (ES11): Represents large integers that can't be represented by the Number type.

const bigIntValue = 1234567890123456789012345678901234567890n;


Non-Primitive (Reference) Data Types:

Object: Represents a collection of key-value pairs.

{"let person = {"}

  firstName: "John",

  lastName: "Doe",

  age: 30

{"};"}

Array: Represents a collection of elements, indexed by numbers.

let colors = ["red", "green", "blue"];

Function: A block of reusable code.

{"function greet(name) {"}

  return `Hello, ${name}!`;

{"}"}

Date: Represents dates and times.

let currentDate = new Date();

RegExp: Represents regular expressions for pattern matching.

let pattern = /[a-z]+/g;

Map (ES6): A collection of key-value pairs with unique keys.

let myMap = new Map();

myMap.set('key1', 'value1');

Set (ES6): A collection of unique values.

let mySet = new Set([1, 2, 3, 4, 4, 4]);

ArrayBuffer (ES6): Represents a fixed-length binary data buffer.

let buffer = new ArrayBuffer(16);

Typed Arrays (ES6): Represents arrays of a specific type of binary data.

let intArray = new Int32Array(buffer);

Error: Represents an error object that can be thrown using the throw statement.

throw new Error('This is an error message');

These are the main data types in JavaScript. Understanding these types is crucial for effective programming in JavaScript.
