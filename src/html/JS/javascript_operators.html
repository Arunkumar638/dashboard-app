Operators

JavaScript operators are symbols or keywords that allow you to perform operations on operands. Operands can be variables, values, or expressions. There are several types of operators in JavaScript:


Arithmetic Operators:

These operators perform mathematical operations.


Examples:

// Addition Operator

let a = 1 + 2

console.log(a);

// Subtraction Operator

let b = 10 - 7

console.log(b);

// Multiplication Operator

let c = 3 * 3

console.log(c);

// Division Operator

let d = 1.0 / 2.0

console.log(d);

// Modulas Operator

let e = 9 % 5

console.log(e)

// Exponentian Operator

let f = 2 ** 5

console.log(f)

// Increment Operator

let g = 2;

g1 = g++;

console.log(g)

// Decrement Operator

let h = 2;

h1 = h--;

console.log(h)

// Unary plus Operator

let i = 3;

i1 = +i;

console.log(i1)

// Negation Operator

let j = 3;

j1 = -j;

console.log(j1)


Assignment Operators:

These operators are used to assign values to variables.

Examples:

let a = 10;

let b = 5;

// Simple assignment

let c = a; // c = 10

// Addition assignment

a += b; // equivalent to a = a + b; a = 15

// Subtraction assignment

a -= b; // equivalent to a = a - b; a = 10

// Multiplication assignment

a *= b; // equivalent to a = a * b; a = 50

// Division assignment

a /= b; // equivalent to a = a / b; a = 10

// Modulus assignment

a %= b; // equivalent to a = a % b; a = 0


Comparison Operators:

These operators compare two values and return a boolean result.

Examples:

let x = 10;

let y = 5;

// Equal to

let isEqual = x == y; // isEqual = false

// Not equal to

let isNotEqual = x != y; // isNotEqual = true

// Greater than

let isGreater = x > y; // isGreater = true

// Less than

{"let isLess = x < y; // isLess = false"}

// Greater than or equal to

let isGreaterOrEqual = x >= y; // isGreaterOrEqual = true

// Less than or equal to

{"let isLessOrEqual = x <= y; // isLessOrEqual = false"}


Logical Operators:

These operators are used to perform logical operations on boolean values.

Examples:

let a = true;

let b = false;

// AND

let andResult = a && b; // andResult = false

// OR

let orResult = a || b; // orResult = true

// NOT

let notResult = !a; // notResult = false

Unary Operators:

These operators operate on a single operand.

Examples:

let x = 10;

// Unary plus (+)

let positive = +x; // positive = 10

// Unary minus (-)

let negative = -x; // negative = -10

// Increment (++)

x++; // x = 11

// Decrement (--)

x--; // x = 10


String Operators:

The + operator is used to concatenate strings.

Example:

let firstName = "John";

let lastName = "Doe";

let fullName = firstName + " " + lastName; // fullName = "John Doe"


Conditional (Ternary) Operator:

This operator evaluates a condition and returns one of two values depending on whether the condition is true or false.

Example:

let age = 20;

{"let canVote = (age >= 18) ? "Yes" : "No"; // canVote = "Yes""}

typeof: Returns the type of a variable or expression.

instanceof: Returns true if an object is an instance of a specific object type.

Bitwise Operators:

These operators perform bit-level operations.

Example:

let x = 6;  // Binary representation: 0101

let y = 1;  // Binary representation: 0011

let result = x & y;  // Bitwise AND

console.log(result);

let result1 = x | y;  // Bitwise OR

console.log(result1);

let result2 = x ^ y;  // Bitwise XOR

console.log(result2);

let result3 =  ~ x;  // Bitwise NOT

console.log(result3);

let result4 =  x >> y;  // Bitwise Right Shift

console.log(result4);

{"let result5 =  x << y;  // Bitwise  Left Shift"}

console.log(result5);

{"let result6 =  x >>> y;  // Bitwise Zero Fill Right Shift"}

console.log(result6);


These are the main types of operators in JavaScript, each serving a specific purpose. Keep in mind that understanding how operators work is fundamental to writing efficient and effective JavaScript code.
