Using state and effect hooks

Using the useState and useEffect hooks in React allows you to add state and perform side effects in functional components. Here's an example of how to use these hooks.


Using useState:

import React, { useState } from 'react';

const Counter = () => {

  // Declare a state variable named "count" with an initial value of 0

  const [count, setCount] = useState(0);

  // Event handler to increment the count

  const increment = () => {

    setCount(count + 1);

  };

  return (

     <div>

       <p>Count: {count} </p>

       <button onClick={increment}>Increment </button>

     </div>

  );

};

export default Counter;


In this example:
  • We import useState from React.
  • Inside the functional component Counter, we use useState(0) to declare a state variable named count with an initial value of 0.
  • We also declare a function setCount that allows us to update the count state.

    • Using useEffect:

    import React, { useState, useEffect } from 'react';

    const Timer = () => {

      const [seconds, setSeconds] = useState(0);

      useEffect(() => {

        const interval = setInterval(() => {

          setSeconds(prevSeconds => prevSeconds + 1);

        }, 1000);

        // Cleanup the interval on component unmount

        return () => clearInterval(interval);

      }, []); // Empty dependency array means this effect runs once on component mount

      return  <div>Seconds: {seconds} </div>;

    };

    export default Timer;


    In this example:
  • We import useState and useEffect from React.
  • We declare a state variable seconds and a function setSeconds with an initial value of 0.
  • In the useEffect hook, we set up an interval that increments seconds by 1 every second.
  • We also return a cleanup function in useEffect to clear the interval when the component is unmounted.
