d.React.lazy

React.lazy is a function introduced in React 16.6 that allows you to dynamically import components. This is particularly useful for implementing code splitting in React applications.


Here's how React.lazy works:

  • It enables you to dynamically load a component only when it's needed, rather than including it in the initial bundle.
  • It returns a component that will automatically load the target component (via code splitting) when rendered.
  • It's typically used in combination with React's Suspense component to handle loading states.

    • Here's an example of how to use React.lazy:

    import React, { Suspense } from 'react';

    const LazyComponent = React.lazy(() => import('./LazyComponent'));

    const App = () => {

    return (

    <div>

    <Suspense fallback={ <div>Loading... </div>}>

    <LazyComponent />

    </Suspense>

    </div>

    );

    };

    export default App;


    In this example:

    React.lazy(() => import('./LazyComponent')) dynamically imports the LazyComponent. It will be loaded on-demand when LazyComponent is first rendered.

    <Suspense> is a component provided by React that allows you to specify a fallback content to display while the lazy component is loading. In this case, we're showing a "Loading..." message.


    Using React.lazy along with Suspense simplifies the process of implementing code splitting in your React application, allowing you to load components on-demand and improve performance by reducing the initial bundle size.
