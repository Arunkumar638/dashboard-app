Making routes dynamic using route params in react

In React Router, you can make routes dynamic by using route parameters, also known as URL parameters or path variables. This allows you to pass data through the URL and use it in your components.

Here's how you can make routes dynamic using route params:


Step 1: Define a Dynamic Route

In your App.js (or wherever you define your routes), specify a dynamic route with a parameter using :paramName.

import React from 'react';

import { BrowserRouter as Router, Route, Switch } from 'react-router-dom';

import Home from './Home';

import UserProfile from './UserProfile';

import NotFound from './NotFound';

const App = () => {

  return (

    <Router>

      <Switch>

        <Route exact path="/" component={Home} />

        <Route path="/user/:username" component={UserProfile} />

        <Route component={NotFound} />

      </Switch>

    </Router>

  );

};

export default App;

In this example, username is a dynamic parameter that can be any value. It will be available in the UserProfile component.


Step 2: Access Route Params in Component

In your UserProfile component, you can access the route parameter using the useParams hook (or withRouter for class components).

import React from 'react';

import { useParams } from 'react-router-dom';

const UserProfile = () => {

  const { username } = useParams();

  return <div>User Profile for: {username}</div>;

};

export default UserProfile;

In this example, useParams is a hook provided by React Router that allows you to access the route parameters. It returns an object containing the parameter values.


Step 3: Link to Dynamic Route

To link to a dynamic route, you can use the Link component and include the parameter value in the to prop.

import React from 'react';

import { Link } from 'react-router-dom';

const UsersList = () => {

  return (

    <ul>

      <li><Link to="/user/johndoe">John Doe</Link></li>

      <li><Link to="/user/janesmith">Jane Smith</Link></li>

    </ul>

  );

};

export default UsersList;


Description

In this example, clicking on the links will navigate to the dynamic route with the corresponding username.

Dynamic routes can be useful for creating pages that display specific content based on the URL parameters.

By using dynamic route parameters, you can create more flexible and interactive web applications that respond to changes in the URL. This is particularly useful for applications with user-specific content or data.
