Pointers and Strings with Examples

In C, strings are represented as arrays of characters, terminated by a null character ('\0'). Pointers are used extensively with strings for various operations. Here are some examples to illustrate the relationship between pointers and strings:
String as an Array:
#include <stdio.h>
int main() {
char str[6] = {'H', 'e', 'l', 'l', 'o', '\0'};
char *ptr = str; // 'ptr' points to the first character of 'str'
printf("%s", ptr); // Output: Hello
return 0;
}
In this example, str is an array of characters that forms the string "Hello". We can use a pointer ptr to point to the first character of str. Then, we use %s in the printf() function to print the string.
String Initialization Using a Pointer:
#include <stdio.h>
int main() {
char *ptr = "Hello"; // 'ptr' is a pointer to a string literal
printf("%s", ptr); // Output: Hello
return 0;
}
In this example, ptr is a pointer to a string literal "Hello". String literals are automatically null-terminated, so there's no need to explicitly include the null character.
Passing Strings to Functions:
#include <stdio.h>
void printString(char *str) {
printf("%s", str);
}
int main() {
char myString[] = "Hello"; // 'myString' is an array of characters
printString(myString);
return 0;
}
In this example, we pass the array myString to the function printString. The function takes a pointer to the first character of the string. Inside the function, we can use %s to print the string.
Using scanf() to Read a String:
#include <stdio.h>
int main() {
char name[20]; // Allocate space for a string
printf("Enter your name: ");
scanf("%s", name); // 'name' is already a pointer to the first character
printf("Hello, %s!", name);
return 0;
}
In this example, we use scanf() to read a string input from the user into the array name.
Since name is an array, it already decays into a pointer to the first character when passed to scanf().
These examples demonstrate how pointers and strings work together in C. Understanding how to use pointers with strings is important for tasks like string manipulation, input/output operations, and working with text data. Remember to be careful with string operations to avoid buffer overflows and other potential issues.
