File handling

File handling in C allows you to perform operations on files like creating, reading, writing, and closing them. To work with files in C, you'll need to include the stdio.h header file, which contains functions like fopen, fclose, fprintf, fscanf, and more. Here are some basic file operations in C:
1. Opening a File (fopen):
#include <stdio.h>
int main() {
FILE *filePtr; // File pointer
filePtr = fopen("example.txt", "w"); // Opens a file in write mode
if (filePtr == NULL) {
printf("Error opening file.\n");
return 1;
}
fprintf(filePtr, "This is a test.\n");
fclose(filePtr);
return 0;
}
In this example, a file named example.txt is opened in write mode ("w") using fopen. If the file can't be opened, it returns a NULL pointer. If successful, it writes "This is a test." to the file and then closes it using fclose.
2.Reading from a File (fscanf):
#include <stdio.h>
int main() {
FILE *filePtr;
filePtr = fopen("example.txt", "r"); // Opens the file in read mode
if (filePtr == NULL) {
printf("Error opening file.\n");
return 1;
}
char buffer[100];
fscanf(filePtr, "%s", buffer); // Reads a string from the file
printf("Read from file: %s\n", buffer);
fclose(filePtr);
return 0;
}
This example opens the example.txt file in read mode ("r") using fopen. It then uses fscanf to read a string from the file and prints it.
3. Appending to a File ("a" Mode):
#include <stdio.h>
int main() {
FILE *filePtr;
filePtr = fopen("example.txt", "a"); // Opens the file in append mode
if (filePtr == NULL) {
printf("Error opening file.\n");
return 1;
}
fprintf(filePtr, "Appending some text.\n");
fclose(filePtr);
return 0;
}
Here, the file is opened in append mode ("a"), which allows you to add content to the end of the file. It appends "Appending some text." to the file.
4. Checking for End-of-File (EOF):
#include <stdio.h>
int main() {
FILE *filePtr;
filePtr = fopen("example.txt", "r");
if (filePtr == NULL) {
printf("Error opening file.\n");
return 1;
}
char buffer[100];
while (fscanf(filePtr, "%s", buffer) != EOF) {
printf("%s ", buffer);
}
fclose(filePtr);
return 0;
}
This program reads words from the file until it reaches the end of the file (EOF). It checks for EOF as the return value of fscanf.
5. Error Handling:
Always check if the file opening was successful by comparing the file pointer to NULL. This ensures that you don't attempt operations on a non-existent or inaccessible file.
Remember to close files using fclose when you're done with them to free up system resources.
These are some basic file handling operations in C. File handling is an important topic, and there are more advanced operations and error handling techniques to learn as you progress.
