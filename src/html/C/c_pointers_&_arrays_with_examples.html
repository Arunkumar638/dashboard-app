Pointers & Arrays with Examples

Pointers and arrays are closely related in C. In fact, an array name itself is essentially a pointer to the first element of the array. Here are some examples to illustrate the relationship between pointers and arrays:
Pointer to Array:
#include <stdio.h>
int main() {
int arr[5] = {1, 2, 3, 4, 5};
int *ptr = arr; // 'ptr' points to the first element of 'arr'
for(int i = 0; i < 5; i++) {
printf("%d ", *(ptr + i)); // Access elements using pointer arithmetic
}
return 0;
}
In this example, arr is an array of integers. When we declare int *ptr = arr;, we're making ptr point to the first element of arr. We can use pointer arithmetic to access the elements of the array.
Array of Pointers:
#include <stdio.h>
int main() {
int num1 = 10, num2 = 20, num3 = 30;
int *ptrArr[3] = {&num1, &num2, &num3}; // 'ptrArr' is an array of integer pointers
for(int i = 0; i < 3; i++) {
printf("%d ", *(ptrArr[i])); // Access elements using pointers stored in the array
}
return 0;
}
In this example, ptrArr is an array of integer pointers. Each element of ptrArr holds the address of an integer variable (num1, num2, num3). We can use *(ptrArr[i]) to access the values stored in these variables.
Pointer Arithmetic with Arrays:
#include <stdio.h>
int main() {
int arr[5] = {1, 2, 3, 4, 5};
int *ptr = arr; // 'ptr' points to the first element of 'arr'
for(int i = 0; i < 5; i++) {
printf("%d ", *(ptr + i)); // Access elements using pointer arithmetic
}
return 0;
}
In this example, we're using pointer arithmetic to access elements of the array. *(ptr + i) allows us to access the i-th element of the array.
Passing Arrays to Functions:
#include <stdio.h>
void printArray(int *arr, int size) {
for(int i = 0; i < size; i++) {
printf("%d ", arr[i]); // Access elements using array notation
}
}

In this example, we pass the array arr to the function printArray. The function takes a pointer to the first element of the array and the size of the array. Inside the function, we can access elements using array notation.
These examples demonstrate the close relationship between pointers and arrays in C.
Understanding how to use pointers with arrays is crucial for tasks like dynamic memory allocation and efficient manipulation of data structures.
