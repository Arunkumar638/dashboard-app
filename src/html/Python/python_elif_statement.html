The elif statement

The elif statement enables us to check multiple conditions and execute the specific block of statements depending upon the true condition among them. The elif statement works like an if-else-if ladder statement in C. It must be succeeded by an if statement.


Control Flow

Syntax

if expression 1:

  # block of statements

elif expression 2:

  # block of statements

elif expression 3:

  # block of statements

else:

  # block of statements


Example

num = 10

if (num == 0):

  print(“Number is Zero”)

elif (num > 5):

  print(“Number is greater than 5”)

else:

  print(“Number is smaller than 5”)
