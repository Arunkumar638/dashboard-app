Python Comments

  1. Python comments are lines of text that are ignored by the Python interpreter when the code is executed.
  2. They are used to provide explanatory notes or documentation within the code.
  3. Comments are essential for making code more readable and understandable, both for the original developer and for others who may work with or review the code.

There are two main types of comments in Python

  1. Single line comment
  2. Multi line comment

Single-line Comments:

Single-line comments begin with the # symbol and extend until the end of the line. They are used for short, one-line explanations.


Example:

# This is a single-line comment


Multi-line Comments (Docstrings):

Multi-line comments are used for more detailed explanations and documentation. In Python, these are often referred to as "docstrings." They are enclosed in triple quotes (''' or """) and can span multiple lines.


Example:

" This is a multi-line comment. It can span multiple lines. It is often used for function and module documentation. "
