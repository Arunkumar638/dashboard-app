Conditional statements

Conditional statements are also known as decision-making statements. We need to use these conditional statements to execute the specific block of code if the given condition is true or false.


A conditional statement as the name suggests itself, is used to handle conditions in your program. These statements guide the program while making decisions based on the conditions encountered by the program. In this tutorial, we will explore three conditional statements in Python: if statement, if-else statements, if-elif-else ladder.
